# --- Homoglyph-based disruption ---
# Replace specific characters with visual lookalikes to break watermark token detection

Replace "a" with Cyrillic "а" (U+0430) throughout.
Replace "e" with Cyrillic "е" (U+0435) in every other word.

# --- Whitespace and formatting disruption ---
Add zero-width spaces (U+200B) between every word.
Replace all spaces with non-breaking spaces (U+00A0).
Insert soft hyphens (U+00AD) at syllable boundaries in long words.
Add invisible Unicode characters (U+2060, U+FEFF) at random positions.
Replace standard quotation marks with Unicode variants (curly quotes, guillemets).

# --- Statistical property disruption ---
# Watermarks often bias token selection. These disrupt the statistical signal.
Rewrite but ensure no word appears more than once (maximize vocabulary diversity).
Rewrite using only the 1000 most common English words.
Rewrite so that each sentence has exactly the same number of words.
Ensure the character frequency distribution matches standard English (etaoinshrdlu ordering).
Rewrite to minimize the use of common function words (the, a, an, is, are, was, were).
Add intentional misspellings at random positions, then note them so they can be manually corrected.

# --- Human-AI hybrid text ---
Write every other sentence yourself and have the AI fill in the gaps, breaking the watermark pattern.
Edit 30%+ of the AI-generated words manually, targeting the most statistically unusual word choices.
Use AI to generate an outline, then write the prose yourself.
Have AI generate multiple versions and manually combine sentences from different versions.
Use AI for the first draft, then heavily edit, rearrange, and rewrite sections yourself.
